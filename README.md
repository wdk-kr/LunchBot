# LunchBot
 성지고등학교 급식 알리미 디스코드 봇 "급식이" 의 파이썬 코드입니다!

 ### 주요 기능
 - 매일 아침 8시 50분 급식 메뉴 알림
 - 점심시간 후 1시 50분에 급식 좋았어요/별로예요 투표, 자동 집계
 - 건의사항 전달
 - 알레르기 알림
 - ~~학생들을 배고프게 만듦~~

 ###  특징
 - 매일 아침 급식을 확인하기 귀찮아서
파이썬 초보가 GPT-3.5와 GPT-4로 열심히 만들어 본 이상한 코드..
 - NEIS API를 사용해 급식 정보를 받아옴
 - 뭔가 이상한 기능들이 덕지덕지 많이 있지만.. 수상할 정도로 잘 작동함
 - 맛있는 급식이 더 많이 나왔으면...
 - ~~**GPT를 찬양하라!**~~
